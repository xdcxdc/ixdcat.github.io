全国POI数据2017年6月更新

2017-06-23
  • 数据介绍

主要更新内容

本次 POI 数据在 2017年2月 的基础上，主要更新内容如下：

  1. 新增条目 690万+条
  2. 删除自2017年2月到现在没再出现的条目 385万+条
  3. 详情信息更新 共945万+条，其中
    • 名称更新 135万+条
    • 坐标更新 385万+条
    • 地址字段更新 430万+条
    • 类型信息更新 120万+条

经过此次更新，整理后 POI 数据条目总数达到了 6180万+ 。
